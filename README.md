# avm-cctp-contracts

## Build

To install dependencies:

```bash
bun install
```

To compile contracts:

```bash
bunx tealscript MessageTransmitter.algo.ts dist
bunx tealscript TokenMessenger.algo.ts dist
bunx tealscript TokenMinter.algo.ts dist
```
